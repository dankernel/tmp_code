
1. 변수.

변수란 프로세스 내에서 사용되는 변할 수 있는 값이다.
예외) 함수의 인자를 const로 받을 때에는 변하지 않는다.

일반적으로, 지역변수는 스택 영역에, 전역변수는 데이터 영역에 저장된다.
참조) ELF, PE포맷, 스택

2. 선언과 초기화

예) 
a. int num;
b. int size = 10;
c. char ch;
d. char ch = 'D';
e. char ch = '5';

a는 int형 변수 num을 선언 한 것이다. 

2-1. 타입
변수의 종류를 의미한다. 변하는 값이 어떤 형태인지를 정의한다.
정수, 실수, 문자 등의 타입을 정의 해야한다.

모든 변수는 메모리 안에 바이너리(2진수)로 저장되는데,
어떻게 해석 할지를 정의해야 한다.

실제로 숫자 저장하던, 문자를 저장하던 이진수(0101...)로 저장되는데,
2진수 값을 숫자로 해석할지, 문자로 해석할지를 정해야 한다.

int같은 변수의 타입은 정해진 몇몇의 것들(int, char, long, float ...)이 있고
사용자가 재정의 해서 사용 할 수 있다. 

int형은 정수형 변수이며 32비트 크기이다.
long형은 정수형 변수이며, 64비트 크기이다.
char형은 문자형 변수이며, 8비트 크기이다.
float형은 실수형 변수이며, 00비트 크기이다.

2-2. 이름
변수의 이름은 정하는 규칙이 몇가지 있는데, 영문으로 시작하고, 언더바(_)와
하이픈(-), 을 사용 할 수 있으며, 숫자는 첫 문자로 사용 할 수 없다.

b는 int형 변수 size를 선언하고 10으로 초기화한 것이다.
c는 char형 변수를 선언 한 것이고, d는 선언과 함께 'D'로 초기화 한 것이다.
char형은 작은 따옴표(')를 기준으로 문자열 하나(1바이트)만 저장이 가능하다.

e는 '5'로 선언 한 것인데, 숫자 5가 아니라 문자 '5'로 초기화 한 것이다.

3. 출력과 활용

int num = 10;
print("%d \n", num);

char ch = 'D';
print("%c \n", ch);

float f = 3.14;
print("%f \n", f);

출력은 print의 함수로 하며, 큰 따옴포(")안에 있는 첫번째 인자와
쉼표(,)로 구분되는 1개 이상의 인자롤 갖는다.

참고) 가변인자 함수, 함수의 안자 갯수가 지정되지 않은 함수.

int형은 %d, float형은 %f... 와 같다.
이진 바이너리를 해석하여 출력하는 방식이 정수, 소수, 문자 등에 따라 구분되는 것이다.
